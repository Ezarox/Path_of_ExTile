Adjust/restore weights. Example:
  setAiWeights(obj), resetAiWeights()

  sweepAiWeights(seed, samples=50, options={}), tuneAiWeights, tuneAiWeightsForSeeds, sampleWeightsVariant
  sweepAiWeights("318495518", 10, { runs: 1 })
  sweepAiWeights("318495518", 40, {runs: 1, varyOnly: "slowInteraction", ranges: { slowInteraction: 1 }})
  tuneAiWeights("318495518", 8, { range: 1.2 })
  tuneAiWeightsForSeeds(["A","B"], 10, { range: 0.5 })
  tuneAiWeightsForSeeds(["a-mazing-race","a-mazing-game","18112025","3270"], 40, { range: 1, runs: 5, minWeight: 0.01 })
